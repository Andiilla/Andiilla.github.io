About The Project

This project aims to create a comprehensive handbook on interpolation techniques for dealing with missing rainfall data. Rainfall is an important parameter in climate and hydrological analysis, but often the data is incomplete or missing for various reasons. In this guide, various interpolation methods that can be used to fill in missing values ​​in rainfall data will be explained in detail, so that the analysis is more accurate and reliable.
